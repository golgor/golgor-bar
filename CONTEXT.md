# golgor-bar

A personal Quickshell-based desktop shell for Hyprland, replacing Waybar. Keyboard-first, minimal, extensible. Runs on Omarchy Linux.

## Language

**Bar**:
The always-visible top status strip spanning the full monitor width.
_Avoid_: Panel, taskbar

**Module**:
A self-contained functional unit within the bar (e.g. Clock, Workspaces, Tray).
_Avoid_: Widget, plugin, applet

**Popout**:
A floating panel that drops out from the bar or screen edge, anchored to its trigger point with an organic chamfered shape. Toggled via hotkey or click, dismissed via hotkey/Escape/click-outside.
_Avoid_: Blob, dropdown, popup, drawer

**Service**:
A QML singleton that owns a specific slice of system state (e.g. Theme, Time, Battery). Services are consumed by modules but never render UI.
_Avoid_: Provider, store, manager

**Component**:
A reusable UI primitive shared across modules (e.g. styled rectangles, text, animations).
_Avoid_: Widget, element

**Theme Service**:
The single service that owns all style properties — colours and fonts. Reads a generated QML file produced by a shell script that parses Omarchy's theme files.
_Avoid_: Style service, colour service

**GeneratedTheme**:
A QML singleton auto-generated by `scripts/generate-theme.sh` from Omarchy's `colors.toml` and font settings. Gitignored. Never edited manually.

## Relationships

- A **Module** may contain one or more **Popouts**
- A **Module** consumes zero or more **Services** for state
- The **Theme Service** imports the **GeneratedTheme** and exposes it to all **Components** and **Modules**
- **Components** are used by **Modules** but never depend on specific modules
- **Services** never depend on **Modules** or **Components**

## Example dialogue

> **Dev:** "Where do I put the calendar?"
> **Domain expert:** "The calendar is a **Popout** owned by the Clock **Module**. It lives in `modules/bar/popouts/`."

> **Dev:** "Should the WiFi scanner be a **Popout**?"
> **Domain expert:** "No — we delegate to system TUIs. The Network **Module** just shows status and launches `nmtui` in a floating terminal."

> **Dev:** "I want to change the accent colour."
> **Domain expert:** "That comes from the **Theme Service** via the **GeneratedTheme**. Change your Omarchy theme, and `scripts/generate-theme.sh` regenerates it."

## Flagged ambiguities

- "Blob" was considered as an alternative to **Popout** — resolved: use **Popout** for the behavior, the blob-like organic/chamfered aesthetic is a styling concern within the popout component.
